## <a name="cursor"/> 光标类

w 到下一个单词到开始

e 到下一个单词的结束

b 到上一个单词的开始

$ 到行尾

0 到行首

h j k l = 左上下右

## <a name="modify"/> 修改类

### 支持：操作符 [数字] 动作（光标）

d 删除

c 更改（其实就是删除加插入）

### 其他

A 行尾插入

U 对整行修改的撤销

r 单字符的替换（不进入插入模式）

R 进入替换模式，可以连续替换

## <a name="anchor"/> 定位类

### 行定位

CTRL-G 查看当前文件信息及行号

G 到末行

gg 到首行

[n]G 到第n行

### 查找与替换

? 代替/，逆向查找

:set

 - ic 忽略大小写
 - hls 高亮匹配项
 - is 显示已输入字符的第一个匹配项（显示部分匹配）

:s/old/new/g 全行替换，去掉 /g 就是首个字符替换，/gc 就是替换前问一下，冒号后面加 % 就是全文件范围内操作，:[n1],[n2]s 就是对 n1 和 n2 间的行进行操作

## <a name="other"/> 其他

:![command] 执行外部命令 command

:w [name] 另保存为 name

用 v 选定一些文本，: 后输入 w [name]，就可以将选定部分保存下来

:r [name] 插入 name 文件中的内容，name 替换为 ![command] 还可以存下来外部命令的输出
